
# Aim Trainer

A Python-based desktop application built with Pygame designed to track and improve mouse accuracy, reaction times, and tracking speed. 

## Features
* **Custom Game Modes:** Engineered both 'Flick' (reaction-based) and 'Tracking' (continuous velocity) game loops.
* **Dynamic Difficulty Scaling:** Algorithmically scales target radius, movement speed, and spawn intervals based on elapsed time.
* **Data Persistence:** Utilizes JSON file handling to save local user configurations and high-score profiles.
* **Custom Physics Engine:** Features localized collision detection, boundary logic, and real-time trajectory updates using 2D coordinate mathematics.

## Tech Stack
* **Language:** Python 3
* **Libraries:** Pygame, JSON, Time, Dataclasses
